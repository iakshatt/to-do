# ✨ Advanced Animated To-Do List

A modern and responsive To-Do List web application built using **HTML, CSS, and JavaScript** with an attractive glassmorphism interface, smooth animations, task filtering, and local storage support.

---

## 🚀 Features

- ✅ Add new tasks
- 🗑 Delete tasks with animations
- ✔ Mark tasks as completed
- 🔍 Search tasks instantly
- 🎯 Task priority system (High, Medium, Low)
- 📊 Completed & Pending task statistics
- 💾 Local storage support
- ✨ Smooth UI animations
- 🎨 Modern glassmorphism design
- 📱 Fully responsive layout
- 🧹 Clear completed tasks feature

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
